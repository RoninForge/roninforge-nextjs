# Changelog

## [1.0.0] - 2026-04-12

### Added

- 5 rule files: core Next.js 15, anti-patterns, Server Components/Actions, routing, and 4-tier caching
- `/nextjs-page` skill for scaffolding App Router pages with async params and metadata
- `/nextjs-api` skill for generating Route Handlers with correct caching
- `/nextjs-validate` skill for scanning projects for Pages Router and v14 patterns
- `/nextjs-migrate` skill for Pages Router to App Router and v14 to v15 migration
- `nextjs-reviewer` subagent for automated code review
- Test fixtures with anti-pattern and correct-pattern samples
- Plugin validation script and CI workflow
